# Giphy App
 
## Goals
- Build a clone of a giphy app that has both frontend and backend technologies
- Allow users to search for gifs

## Technologies
- React
- Node.js
- Javascript
- HTML 5
- CSS
- Bootstrap 4
- Axios
- Giphy API

## Features List
- Displays trending gifs as a default homepage
- Search random gifs
